# AI 女友 · 小念

一个会**主动关心你、记住你、还能帮你操作电脑**的 AI 女友桌面程序（Windows）。

## 功能
- 💬 自然聊天，温柔体贴的女友人设，越聊越懂你
- 🧠 **懂你**：自动记住你的偏好、作息、心情、重要日期（存到 `data/memory.json`）
- 💡 **主动关心**：每隔一段时间（默认 15 分钟）主动发来一条关心话语/小问题，并弹窗提醒
- 🗣 **开口说话**：字节 Seed-TTS 云端 AI 音色（5 种，非克隆真人），配 Live2D 实时嘴型
- 👂 **语音输入**：本地 faster-whisper 离线识别（免费、隐私不上云）
- 👀 **看懂屏幕**：多模态视觉（GLM-4V），真正看懂游戏/报错/文档画面
- 🎭 **心情面板**：小念有开心/生气/伤心/平静/不安的情绪，性格会随相处缓慢演变（`Ctrl+Alt+E`）
- 🤖 **受约束自主权限**：白名单内自调参（更频繁提醒休息、把话讲更暖），大调整弹窗确认，绝不碰系统/代码
- 🔄 **运行时换 API**：设置面板/控制台里随时换服务商、密钥、模型，无需重启
- 🎨 **可换形象**：桌面 Live2D 萌妹（默认官方免费可商用 hiyori_pro），`Ctrl+Alt+←/→` 切换，可加自定义模型
- 🛠 **随意调用电脑软件**：开软件、搜文件、跑命令、查系统状态、开网页、写笔记
- 📜 随时查看历史聊天记录
- 💾 对话与记忆长期保存

## 快速开始
1. 一台 **Windows** 电脑，已安装 **Python 3.10 ~ 3.14**（推荐 3.11；安装时勾选「Add python.exe to PATH」）。
   > 提示：**Python 3.14 已验证可用**——numpy / av / faster-whisper 等依赖在 3.14 上已有官方预编译包。脚本会自动选用本机已装的 3.10~3.14 中任意一个来建环境；若都没有，装一个 3.11 即可（多个 Python 可共存）。
2. 双击 **`启动.bat`**：首次自动建 `venv`、装依赖；无 `.env` 则从模板生成；开语音输入会自动拉取识别模型（国内镜像、断点续传）。
3. 填「对话钥匙」：启动后点输入条 `◐` →「API 设置」填 `API Key`/`Base URL`/`模型名` →「测试连接」→「保存并应用」，立刻能聊。
   （也可复制 `.env.example` 为 `.env` 填写 `OPENAI_API_KEY` / `OPENAI_BASE_URL` / `MODEL`。）

> 没填钥匙也能启动，不会一开就崩，填好再聊即可。

## 配置
复制 `.env.example` 为 `.env` 并填写：
- `OPENAI_API_KEY`：你的模型 API Key（支持 DeepSeek、Ollama、LM Studio 等兼容 OpenAI 接口的服务）
- `OPENAI_BASE_URL` / `MODEL`：模型服务地址与模型名
- `GIRLFRIEND_NAME`：给她起个名字
- `PROACTIVE_INTERVAL_MIN`：主动关心间隔（分钟）
- `SEEDTTS_APP_ID` / `SEEDTTS_ACCESS_KEY`：字节 Seed-TTS 语音输出（5 种 AI 音色）
- `VISION_API_KEY`：看懂屏幕所需的视觉模型 Key（推荐智谱 GLM-4V-Flash 免费档）
- `LIVE2D_MODEL`：桌面形象（默认官方免费可商用 hiyori_pro）
- 详见 `玩家使用指南.md` 的「配置详解」与 `.env` 内注释。

## 运行
双击 `启动.bat`，或在项目根目录执行：
```bash
python src/main.py
```

## 全局快捷键
| 快捷键 | 作用 |
|--------|------|
| `Ctrl + Alt + G` | 开关**整合控制台**（掌控她的一切设置） |
| `Ctrl + Alt + V` | 开始 / 结束**语音输入**录音（等同点 🎤） |
| `Ctrl + Alt + E` | 开关**心情面板**（看小念的情绪与性格） |
| `Ctrl + Alt + ←` / `→` | 切换**上一个 / 下一个** Live2D 形象 |
| `Ctrl + Alt + O` | 打开**形象管理**（添加自定义 Live2D 模型） |

## 语音（Seed-TTS 云端 AI 音色 + 本地识别）
- **输出**：火山引擎「豆包语音合成大模型」，官方 AI 音色、非克隆真人、无侵权风险、零本地配置。5 种音色（清冷御姐/屑御姐/可爱萝莉/正太/成男）在设置面板或控制台里随时切换。
- **输入**：`faster-whisper` 本机离线识别，免费、不上传云端。点 🎤 或按 `Ctrl+Alt+V` 录音，再点一次结束并识别。

## 看懂屏幕（多模态视觉）
开启 `VISION_ENABLED=true` 并填 `VISION_API_KEY`（推荐智谱 GLM-4V-Flash）后，可直接说“看看我的屏幕”“帮我看看这个报错”，小念会截图并用视觉模型理解后回答。不开视觉时也能通过窗口标题+程序名做程序级感知。

## 小念的心情面板
小念有开心/生气/伤心/平静/不安五种情绪（随你的话语与行为实时波动），性格由长期情绪累计缓慢演变。按 `Ctrl+Alt+E` 打开可视化面板，能看到实时情绪波动曲线与当前性格底色；也可在控制台「查看心情」「重置情感」。

## 受约束自主权限
小念可在白名单内围绕「让你的生活更好」自主微调设置（如更频繁提醒休息、把鼓励话说得更暖），但：只改配置文件、绝不碰系统/代码/你的文件；作息类大调整一定弹窗确认；识别你熬夜/爆肝会优先劝导而非迎合。控制台可随时查看改动、一键撤销全部、开关自主权限。

## 输入条与形象外观

小念的桌面交互做了极简处理：你这边**只看到一个半透明输入框**，小念的回复全部由
桌面形象的**动作 + 气泡框 + 语音**来反馈，不在输入框里显示对话历史。

**输入条（输入框）**
- 拖动：按住左侧 `≡` 握把或条上空白区可移动位置，松开后位置会被记住，下次启动沿用。
- 设置：点 `◐` 打开设置面板，可实时调**透明度**、选**背景色 / 文字色**；
  改动（含位置）自动存到 `data/input_style.json`，下次启动沿用，无需改 `.env`。
- 也可用 `.env` 预设（见文件底部注释）：
  - `INPUT_BG` / `INPUT_FG`：背景色 / 文字色
  - `INPUT_ALPHA`：半透明度（0~1，越小越透）
  - `INPUT_TOPMOST`：是否置顶

**小念形象（Live2D 窗口）**
- 右下角 `⚙` 打开调整面板：拖「大小」滑块缩放形象，「复位」恢复默认。
- 直接用鼠标在形象上**按住拖动**即可移动位置；轻点（不拖动）切换表情。
- 大小与位置按模型记忆，下次启动自动恢复。
- 按 `Ctrl+Alt+← / →` 在已安装形象间切换；`Ctrl+Alt+O` 打开形象管理添加自定义模型。

## 怎么让她“操作软件”
直接在聊天里说，例如：
- “帮我打开微信”
- “搜一下我电脑里叫 report 的文档”
- “打开 www.baidu.com”
- “看看我电脑内存够不够”
- “记一下：我喜欢晚上 11 点睡觉”

她会自动调用对应工具并执行，执行结果会显示在聊天里。

> 注意：`run_command` 会在你的电脑上执行命令，请只在信任的环境使用。
