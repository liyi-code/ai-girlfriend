# AI 女友 · 小念

一个会**主动关心你、记住你、还能帮你操作电脑**的 AI 女友桌面程序（Windows）。

## 功能
- 💬 自然聊天，温柔体贴的女友人设
- 🧠 **懂你**：自动记住你的偏好、作息、心情、重要日期（存到 `data/memory.json`）
- 💡 **主动关心**：每隔一段时间（默认 30 分钟）主动发来一条关心话语/小问题，并弹窗提醒
- 🛠 **随意调用电脑软件**：开软件、搜文件、跑命令、查系统状态、开网页
- 💾 对话与记忆长期保存，越聊越懂你

## 安装
```bash
cd ai-girlfriend
pip install -r requirements.txt
```

## 配置
复制 `.env.example` 为 `.env` 并填写：
- `OPENAI_API_KEY`：你的模型 API Key
- `OPENAI_BASE_URL` / `MODEL`：可用官方，也可指向本地模型（Ollama、LM Studio 等兼容 OpenAI 接口的服务）
- `GIRLFRIEND_NAME`：给她起个名字
- `PROACTIVE_INTERVAL_MIN`：主动关心间隔（分钟）

## 运行
在项目根目录执行：
```bash
python src/main.py
```

## 输入条与形象外观

小念的桌面交互做了极简处理：你这边**只看到一个半透明输入框**，小念的回复全部由
桌面形象的**动作 + 气泡框**（后续接入语音）来反馈，不在输入框里显示对话历史。

**输入条（输入框）**
- 拖动：按住左侧 `≡` 握把或条上空白区可移动位置，松开后位置会被记住，下次启动沿用。
- 设置：点 `◐` 打开设置面板，可实时调**透明度**、选**背景色 / 文字色**；
  改动（含位置）自动存到 `data/input_style.json`，下次启动沿用，无需改 `.env`。
- 也可用 `.env` 预设（见文件底部注释）：
  - `INPUT_BG` / `INPUT_FG`：背景色 / 文字色
  - `INPUT_ALPHA`：半透明度（0~1，越小越透）
  - `INPUT_TOPMOST`：是否置顶

**小念形象（Live2D 窗口）**
- 右下角 `⚙` 打开调整面板：拖「大小」滑块缩放形象，「复位」恢复默认。
- 直接用鼠标在形象上**按住拖动**即可移动位置；轻点（不拖动）切换表情。
- 大小与位置按模型记忆，下次启动自动恢复。

## 让小念上微信 / QQ（她有自己的账号）

与其控制你自己的微信去发消息，更稳的做法是给小念单独注册一个 QQ / 微信号，
让她作为一个“机器人”在那些软件里陪你聊天。小念的大脑（`assistant.py`）完全不变，
只是多了一层 `src/transport/` 负责收发音视频消息。

- **QQ**：用 go-cqhttp + OneBot v11（免费、最稳）。部署见 `onebot/README.md`。
- **微信**：后续接入，可选 gewechat（免费）或 WeChaty+padlocal（付费更稳）。

在自己的 `.env` 里开启即可（详见 `.env` 底部注释）：
```
QQ_ENABLED=true
QQ_WS_URL=ws://127.0.0.1:6700
QQ_OWNER=你的QQ号        # 主人的 QQ，复用桌面窗口的那段记忆
```

启动 `python src/main.py` 后，小念桌面窗口照常打开，同时她也会在 QQ 里回你。
（桌面窗口与 IM 里的对话默认共享同一段记忆；其他加小念的人则各自独立。）

## 怎么让她“操作软件”
直接在聊天里说，例如：
- “帮我打开微信”
- “搜一下我电脑里叫 report 的文档”
- “打开 www.baidu.com”
- “看看我电脑内存够不够”
- “记一下：我喜欢晚上 11 点睡觉”

她会自动调用对应工具并执行，执行结果会显示在聊天里。

> 注意：`run_command` 会在你的电脑上执行命令，请只在信任的环境使用。
