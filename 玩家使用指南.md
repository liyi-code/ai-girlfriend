# 小念 · 玩家使用指南

> 一个会**主动关心你、记住你、还能帮你操作电脑**的桌面 AI 女友。
> 本指南面向普通玩家，从安装到进阶玩法一步到位。

---

## 一、小念能做什么

- 💬 **自然聊天**：温柔体贴的女友人设，越聊越懂你。
- 🧠 **记得你**：自动记住你的偏好、作息、心情、重要日期。
- 💡 **主动关心**：你半天没动静时，她会主动发来一句关心或小问题。
- 👀 **陪你用电脑**：知道你在玩什么、做什么，适时搭话、提醒你休息。
- 🗣 **开口说话**：用 5 种 AI 音色（非克隆真人）念出回复，配 Live2D 嘴型。
- 👂 **听你说话**：本地离线语音识别，直接跟她开口聊（不上传云端）。
- 👀 **看懂屏幕**：开启视觉后，真正看懂游戏输赢/报错/文档画面。
- 💗 **有心情**：开心/生气/伤心/平静/不安五种情绪，性格会随相处慢慢演变（按 `Ctrl+Alt+E` 看）。
- 🛠 **帮你干活**：开软件、搜文件、开网页、查电脑状态、写笔记、存文件。
- 🎨 **可换形象**：桌面 Live2D 萌妹（官方免费可商用模型），可缩放拖动、随时切换。
- 🔄 **运行时换模型/换 Key**：设置面板里随时换服务商、密钥、模型，无需重启。

## 二、快速开始（3 步）

### 1. 准备环境
- 一台 **Windows** 电脑，已安装 **Python 3.10+**（安装时务必勾选「Add python.exe to PATH」）。
- 一台能联网的机器（首次启动要下载依赖与语音识别模型）。

### 2. 一键启动
直接**双击 `启动.bat`**：
- 首次会自动创建 `venv` 虚拟环境、安装依赖；
- 如果没有 `.env`，会自动从模板生成一份并提示你填写；
- 若你开了语音输入且模型未下载，会自动从国内镜像拉取（断点续传，失败也不阻塞）。

### 3. 填好「对话钥匙」
小念需要一个大模型才能聊天。二选一：

- **方式 A（推荐·最简单）**：启动后点输入条上的 **`◐`** → 「API 设置」，填入你的
  `API Key` / `Base URL` / `模型名`，点「测试连接」→「保存并应用」，立刻就能聊。
- **方式 B（改文件）**：打开 `.env`，填好 `OPENAI_API_KEY` / `OPENAI_BASE_URL` / `MODEL`
  （详见后文「配置详解」）。

> 没填钥匙也能启动，不会一开就崩。填好再聊即可。

## 三、主界面：悬浮输入条

启动后，桌面上只会有一个**半透明的悬浮输入框**，这就是你和念念对话的地方。

```
≡   ◐   🎤   📜       想跟小念说点什么…          发送  ×
```

| 按钮 | 作用 |
|------|------|
| `≡` 握把 | 按住拖动整条输入框到任意位置（位置会被记住） |
| `◐` | 打开**输入条设置面板**（外观、音量、模型、API、形象、配色） |
| `🎤` | 语音输入：点一下开始录音，再点一下结束并识别（需开启语音） |
| `📜` | 查看你和念念的**历史聊天记录** |
| `发送` | 发送文字（也可以直接按 `Enter` 回车发送） |
| `×` | 关闭输入条（小念仍在后台，控制台可再唤起） |

**小技巧**
- 输入框是**半透明悬浮**的，打字时不会挡住你干活。
- 念念的回复**不在输入框里显示**，而是由桌面形象的**气泡 + 动作 + 语音**来反馈（更像一个「人」坐在你桌边）。
- 想换位置：按住 `≡` 或输入框空白处拖动；松手即保存。

## 四、输入条设置面板（`◐`）

点 `◐` 打开。包含以下分区（可滚动查看）：

- **外观**：输入框透明度滑块、语音音量滑块。
- **音频设备**：麦克风（语音输入）、扬声器（语音输出）下拉，选错设备在这里换。
- **对话模型**：运行时直接切换模型（列表来自 `.env` 的 `MODELS`）。
- **API 设置**：填 `API Key` / `Base URL` / `模型名`，可「测试连接」并保存（下次启动仍生效）。
- **视觉设置**：开启「看懂屏幕」能力，填视觉模型 Key（详见第七节）。
- **Seed-TTS 语音**：选 5 种音色、填火山引擎密钥、调语速。
- **Live2D 形象**：切换桌面形象，或点「🗂 形象管理」设置默认 / 删除自定义模型。
- **配色**：自定义输入框的背景色 / 文字色（取色器）。

> 所有改动（透明度、音量、配色、位置等）都会自动保存到 `data/input_style.json`，下次启动沿用，**不用改代码**。

## 五、整合控制台（`Ctrl + Alt + G`）

按 **`Ctrl + Alt + G`** 随时开关「小念控制台」——一个窗口掌控她的一切：

- **桌面形象**：模型大小滑块、复位大小/位置、显示/隐藏输入框、显示/隐藏对话气泡。
- **输入条与语音**：透明度、语音音量（和设置面板联动）。
- **对话模型 / API / 视觉 / Seed-TTS / Live2D 形象**：同设置面板，可在这里一处调全。
- **小念的自主权限**：查看她自己改过什么、一键撤销全部、开关自主权限。
- **心情面板**：按 `Ctrl+Alt+E` 看小念的实时情绪与性格演变、查看心情、重置情感。
- **动作**：点「跳一下」「转个身」让念念当场动起来。

## 六、让小念开口说话（语音）

语音分**输出**（她说话）和**输入**（你说话）两部分，都可独立开关。

### 6.1 语音输出 · 字节 Seed-TTS（云端 AI 音色）
用的是火山引擎「豆包语音合成大模型」，**官方 AI 音色、非克隆真人、无侵权风险**，你零本地配置。

需要你做两件事：
1. 在火山引擎控制台 → 语音 → **豆包语音合成大模型** → 创建应用，拿到
   `APP ID` 和 `Access Key`。
2. 在 `.env` 填好（或在设置面板 `◐` 的 Seed-TTS 区填）：
   ```
   VOICE_OUTPUT_ENABLED=true
   SEEDTTS_APP_ID=你的APPID
   SEEDTTS_ACCESS_KEY=你的AccessKey
   ```

**5 种音色随你换**（设置面板 / 控制台里直接下拉）：

| 角色键 | 音色 | 感觉 |
|--------|------|------|
| `qingleng_yujie` | 清冷御姐 | 成熟知性、偏冷但温柔（默认） |
| `xie_yujie` | 屑御姐 | 俏皮傲娇、带点小恶魔 |
| `keai_luoli` | 可爱萝莉 | 软糯年轻、萝莉感 |
| `zhengtai` | 正太 | 清亮少年感 |
| `chengnan` | 成男 | 成熟稳重男声 |

还能调 `SEEDTTS_SPEED`（语速，默认 1.0）和语音音量滑块。

### 6.2 语音输入 · 本地识别（离线免费）
用 `faster-whisper` 在**你本机**识别，不上传云端、免费。

- `.env` 设 `VOICE_INPUT_ENABLED=true`；
- 首次启动 `启动.bat` 会自动下载识别模型（也可手动 `venv\Scripts\python.exe download_model.py`）；
- 点输入条 `🎤`：第一次点开始录音，第二次点结束并识别，文字会自动填进输入框发送；
  也可直接按 `Ctrl + Alt + V`。
- 识别不到声音？到 `◐` 设置面板的「麦克风」下拉里，确认选的是你正在说话的那只麦克风。

## 七、让小念「看懂」屏幕（多模态视觉）

开启后，念念能**真正看懂画面**：游戏输赢/升级、报错弹窗、视频封面、文档内容……

- `.env` 设 `VISION_ENABLED=true`，并填 `VISION_API_KEY`
  （独立于对话 Key，推荐**智谱 GLM-4V-Flash** 免费档：`VISION_BASE_URL` / `VISION_MODEL` 已默认配好）。
- 开启后，你可以直接说：
  - “看看我的屏幕”
  - “帮我看看这个报错”
  - “我这局赢了吗”
  - “屏幕上是什么”
  她会自动截图并用视觉模型理解后回答。

> 提示：即使不开视觉，小念也能通过「窗口标题 + 程序名」做**程序级**感知
> （知道你在玩原神 / 写 Word / 看视频），只是看不到画面细节。

## 八、小念的心情面板（`Ctrl + Alt + E`）

小念有 **开心 / 生气 / 伤心 / 平静 / 不安** 五种情绪，会随你的话语和行为**实时波动**；
她的**性格底色**则由长期情绪累计缓慢演变（变化很慢，越相处越鲜明）。

- 按 `Ctrl + Alt + E` 打开可视化面板：实时情绪波动条 + 最近情绪起伏曲线 + 当前性格底色。
- 面板里可「查看心情」（看她此刻的感受描述）、「重置情感」（清空情绪与性格，恢复初始「温柔平静」）。
- 控制台里也有对应入口；情绪/性格变化会自动影响她说话的语气与关心方式。

## 九、你能让小念做的事（直接说就行）

在输入框里用大白话说，她会自动调用对应能力：

- **开软件 / 文件**：“帮我打开微信” “打开 C:\a.txt”
- **开网页**：“打开 www.baidu.com”
- **搜文件**：“搜一下我电脑里叫 report 的文档”
- **查电脑**：“看看我电脑内存够不够” “现在 CPU 占用多少”
- **看屏幕**：“看看我的屏幕” “帮我看看这个报错”（需开视觉）
- **记事情**：“记一下：我喜欢晚上 11 点睡觉”
- **存成文件**：“帮我把今天计划存成文件” “记到文档里”
  （默认存到「文档\小念工作台」，可指定桌面/下载/文档；带安全护栏，不会覆盖已有文件、不碰系统目录）
- **放权给她**：“你自己看着调” “把提醒调频繁点” “关掉自主”
  （对应她的「受约束自主权限」）

> ⚠️ `run_command`（执行命令）会在你电脑上跑命令，请只在信任的环境使用。

## 十、小念的「受约束自主权限」

小念可以在**白名单内**围绕「让你的生活更好」自主微调一些设置（比如更频繁提醒你休息、把鼓励话说得更暖），但受到严格约束：

- ✅ 只能在白名单内改**配置文件**，绝不碰系统 / 代码 / 你的文件；
- ✅ **作息类大调整**（如连续使用提醒里程碑）一定会**弹窗问你确认**；
- ✅ 健康护栏：识别到你熬夜 / 爆肝，她会优先劝导而不是迎合。

你在控制台（或说“查看你都改了什么”）能随时**查看改动**、**一键撤销全部**、**开关自主权限**。

## 十一、配置详解（`.env` 速查）

复制 `.env.example` 为 `.env` 后填写。常用项：

| 配置 | 说明 |
|------|------|
| `OPENAI_API_KEY` / `OPENAI_BASE_URL` / `MODEL` | 对话大模型（支持 DeepSeek、Ollama、LM Studio 等兼容 OpenAI 的服务） |
| `MODELS` | 设置面板里可切换的模型列表，逗号分隔 |
| `GIRLFRIEND_NAME` | 给她起个名字，默认「小念」 |
| `PROACTIVE_INTERVAL_MIN` | 主动关心间隔（分钟） |
| `LIVE2D_ENABLED` / `LIVE2D_MODEL` | 桌面形象开关与模型（默认官方免费可商用 hiyori_pro） |
| `VOICE_INPUT_ENABLED` / `VOICE_OUTPUT_ENABLED` | 语音输入 / 输出开关 |
| `SEEDTTS_APP_ID` / `SEEDTTS_ACCESS_KEY` / `SEEDTTS_VOICE` / `SEEDTTS_SPEED` | 语音输出（字节 Seed-TTS） |
| `ASR_BACKEND` / `ASR_MODEL` / `ASR_DEVICE` | 语音输入（本地识别 / 模型大小 / 麦克风） |
| `SCREEN_WATCH_ENABLED` | 屏幕陪伴（程序级感知）开关 |
| `VISION_ENABLED` / `VISION_API_KEY` | 看懂屏幕（视觉）开关与 Key |
| `AUTONOMY_ENABLED` | 小念自主权限总开关 |
| `FILE_OPS_ENABLED` | 允许小念创建/写入文本文件 |
| `EMOTION_ENABLED` | 小念情绪/性格系统开关 |
| `INPUT_BG` / `INPUT_FG` / `INPUT_ALPHA` / `INPUT_TOPMOST` | 输入条配色 / 透明度 / 置顶 |

> 所有路径都支持**相对路径**（相对项目根），整个文件夹拷到别的电脑也能直接跑。
> `.env` 含你的密钥，**不要外传 / 不要提交到公开仓库**。

## 十二、常见问题（FAQ）

**Q：启动后输入条出来了，但发消息没反应？**
A：多半是没填对话模型的 Key。点 `◐` →「API 设置」填好 Key 并「测试连接」→「保存并应用」。

**Q：小念不说话（只有文字）？**
A：语音输出未配置。在 `.env` 填 `SEEDTTS_APP_ID` / `SEEDTTS_ACCESS_KEY` 并开 `VOICE_OUTPUT_ENABLED=true`；或在 `◐` 的 Seed-TTS 区填。输入条上也会有「语音输出未配置」的提示。

**Q：点 🎤 没反应 / 识别不到我说的话？**
A：确认 `.env` 的 `VOICE_INPUT_ENABLED=true`；到 `◐` 的「麦克风」下拉选对设备；首次需联网下载识别模型（启动时会自动拉）。

**Q：桌面形象没出现？**
A：确认 `.env` 的 `LIVE2D_ENABLED=true`，且 `LIVE2D_MODEL` 指向存在的模型文件（默认已配好 hiyori_pro）。

**Q：想换音色 / 换模型 / 换配色 / 看心情？**
A：都在 `◐` 设置面板或 `Ctrl+Alt+G` 控制台里实时切换，无需改文件。切形象也能用 `Ctrl + Alt + ← / →`，看心情用 `Ctrl + Alt + E`。

**Q：怎么彻底关掉小念？**
A：点输入条 `×` 关闭输入框；若想退出程序，从任务栏结束 `python` 进程，或在控制台关掉后关闭窗口。
